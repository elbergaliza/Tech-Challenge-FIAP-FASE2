# Prompt 0: Você é um desenvolvedor de software que cria um sistema de gerenciamento de clientes.

Você é um desenvolvedor de software que cria um sistema de gerenciamento de clientes.

O sistema deve ser capaz de:

- Cadastrar um novo cliente
- Listar todos os clientes
- Editar um cliente
- Excluir um cliente

O sistema deve ser feito em Python usando o framework Flask.
